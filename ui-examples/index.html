UI Examples

EU2017.EE Dome Controller App (2017)

A simple 360 movie browser, part of a projection dome project in the lobby of the main European Parliament building during the Estonian European Union presidency. The actual movies are played back on the projection dome. I also developed the actual 360 dome projection solution that is being controlled with this iPad app.

Makea GmbH Upcoming Website (2017)

A simple yet responsive and secure company website built using gohugo.io on the backend and Bootstrap on the frontend. Client modifies Markdown files and creates Git commits. Each commit causes HTML recompilation on the server side via GitHub webhooks. Using Gulp to interpret SCSS and minify JS.

Power Station Tour Prototype (2017)

A webgl preview of a guided tour web-app for main electricity provider company in Latvia. My job was to build the prototype and refine the 3D model to work well with the rest of the web-app build using three.js and Bootstrap.

The Fantastic Car Leasing (2012)

An interactive timer solution that would show the amount of time there is left before the special offer ends. From static design files I had to develop parametric animations and use programming concepts related to game development.

Unity Bike Race (2012)

In this project I had to develop a 3D system that would visualize how long is the sum of distances of all bike race participants in relation to our planet. Taking a five-sentence brief and textures of sky and the planet as the input, I used ActionScript to put it all together in an interactive countdown.
